Scaling notes

The honest path from solo-founder MVP to larger SaaS

Layer Now Later
Database Embedded SQLite for launch speed Postgres or DynamoDB before multi-replica writes
Jobs Persisted audit jobs plus due-monitor execution SQS, RabbitMQ, Kafka, or Lambda workers when volume justifies it
Exports JSON/CSV export objects in local object storage S3 archives, signed URLs, and customer report delivery
Integration style REST, OpenAPI, RPC, WebSockets, and polling SDKs, webhooks, and event-stream consumers
Traffic Nginx proxy and app usage limits Kubernetes HPA, WAF rules, and edge caching
Deployment Render, Docker, Kubernetes, and Vercel paths Preview promotion, managed storage, and custom domains
